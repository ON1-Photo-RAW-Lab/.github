## 🧠 Overview

**ON1 Photo RAW** is a comprehensive photo editing platform that combines RAW image processing, AI-powered enhancement, non-destructive editing, layer-based compositing, and photo organization within a single desktop application.

The software is designed to eliminate the need for multiple editing programs by integrating image browsing, catalog management, advanced retouching, masking, effects, HDR creation, panorama stitching, focus stacking, and creative filters into one streamlined workflow.
